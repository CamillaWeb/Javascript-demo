知识点：
	程序的概念
		生活中：进行一组有顺序的过程，程序
		计算机中：要做这个事情的人不是我们本人，而是电脑，我们要通过编程语言去告诉电脑如何完成这个过程
		通过指令，组合它们的顺序，程序

		语言：国语、英语、日语、韩语。。。
		      Java、C、C++、C#、python、php、JS。。。
	javascript的概念以及和ECMAScript的关系
		B/S架构非常火热，B代表的是浏览器，Javascript是浏览器内置语言
		浏览器成就它
		ES5 和 ES6
		客户端	服务器(nodejs)
		V8引擎
	javascript的特点
		动态性（轻量级）
		弱类型（松散类型）
		跨平台性
 	javascript的语法
 		script标签
 			a.不要写在html标签外面
 			b.不要写在head标签里面
 			c.script标签一定要放在body结束标签的前面
 		type属性设置脚本语言的类型（默认type="text/javascript"）
 		*最推荐的方式一定是html代码和逻辑分离，通过script标签的src属性去引入js
 		*一个script标签只能引入一段js代码
 	html中注释
 		<!--这是一个HTML的注释-->
 	css中注释
 		/*这里是CSS中的注释*/
 	javascript中注释
 		//这里是单行注释
 		/*
 		 * @author:star
 		 * @date:2017-07-26 21:30
 		 * @descript:这是多行注释内容
 		*/
 	变量(variable)
 		概念：可以改变的数量
 		[变量][引用]
 		通过变量去 "引用" 内存中的数据
 		*变量第一次使用一定要通过var关键字来定义
 	程序就是不断的操作数据的一个过程，变量是操作数据的唯一途径
 	变量命名规范
 		a.由字母数字下划线(_)和美元符号($)组成
 		b.不能以数字开头
 		c.补充：变量名称要有含义
 		d.补充：如果遇到由多个英文单词所组成的变量名称，使用驼峰命名法
 		e.补充：变量的首字母一定是小写
 		*它跟css类命名规范相似，但是不同，css中允许出现“-”,child-node
 	数据类型
 		undefiend 没有定义数据类型
 		number 数值数据类型 例如 10或者1或者5.5
 		string 字符串数据类型用来描述文本 例如 "你的姓名"
 		boolean 布尔类型 true|false  不是正就是反，是或者否，有或者没有
 		object 对象类型 复杂的一组描述信息的集合
 		function 函数类型 
 		*可以通过一个关键字得到一个数据的数据类型 typeof
英文单词：
	undefined[未定义]、boolean[布尔]、string[字符串]、number[数值]、type[类型]、 syntax[语法]、error[错误]、exception[异常]、true[真]、false[假]
