知识点：
	循环结构
		break关键字：跳出当前代码结构（switch、while、for）
		continue关键字：结束本次循环，进入下次循环
			*continue代码的前面一定要改变循环变量
		for循环
			把循环三要素通过语法一次性定义好了，让我们程序员更加关注循环体的逻辑
			语法:
				for(定义循环变量;判断循环变量;改变循环变量){
					//循环体里，逻辑代码
				}
			例子：
				for(var date=1;date<=31;date++){
					console.log(date);
				}
			for循环的执行顺序：
				1.定义循环变量  只执行一次
				2.判断循环变量
					a.条件成立则执行循环体中逻辑代码，执行完毕逻辑代码之后，最后执行 改变循环变量，又执行步骤2
					b.条件不成立则循环结束
			for循环注意事项：
				1.定义循环变量的指令块中可以定义多个变量
				2.改变循环变量的指令块中可以改变多个变量
				3.判断循环变量的指令块中可以判断多个变量
				3.单纯从语法上来说，for循环中的三个指令块（循环三要素）都可以写在别处，但是一定要加两个分号，不过没含义

		嵌套for循环
			for(var i=0;i<10;i++){
				for(var j=0;j<10;j++){
					
				}
			}
			嵌套循环，整个代码执行的次数是外层循环的次数乘以内层循环的次数
			* 循环次数越多，性能越低
			* 避免多层嵌套循环，通常嵌套循环不超会2层
			* 外层的循环通常控制的是行数，内层的循环通常控制的是列数
			document.write 通过文档流，往body中写text
			
		数组
			由一对中括号所组成，里面每一个元素用逗号分开
			例子：
				var arr = ["衣服","裤子","内裤","袜子"];
			是一块连续的内存空间，它通过下标（索引）来访问对应的内存空间，每一个空间我们都称之为一个数组元素
			* 索引从0开始
			* 可以通过数组对象的length属性获得当前数组元素的个数
			* 遍历的时候要记住，最后一个索引一定是数组length-1
			* 数组元素可以是不同的数据类型
			* 数组也是个对象
